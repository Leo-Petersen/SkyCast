# SkyCast

Weather app for CSCI 322. Shows current weather for Missoula, Montana by default, with the option to use your browser location or search any city. Weather data comes from the Weatherstack API and gets cached in Firebase Firestore.

Best viewed on desktop.

## Live Site

(https://skycastweatherproject.netlify.app/)

## Pages

- Home: current weather for Missoula (or your location)
- Search: look up any city
- Saved Locations: save and manage favorite cities
- About: project info

## Built With

- React / Vite
- React Router
- Firebase Firestore
- Weatherstack API
- Netlify
